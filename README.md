This is a Next.JS Saas-Boilerplate that contains the following features:

- [x] Authentication (NextAuth.js)
- [x] Database (Prisma MongoDB)
- [x] Serverless Functions
- [x] Light/Dark Mode
- [x] Tailwind CSS
- [x] Shadcn ui
- [x] Landing Page
- [x] Translation (i18n)
- [x] Stripe Integration
- [x] Email Integration
